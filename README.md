# GADGON – Power the Future 🚀

A premium, mobile-first online gadget store with Instagram-style grid and WhatsApp ordering.

## 🌐 Live Site
Visit: [gadgon.vercel.app](https://gadgon.vercel.app)

## ✨ Features
- Instagram-style 2-column product grid
- Vertical poster product images (3:4 ratio)
- **WhatsApp ordering** — no cart, just chat!
- Reel/swipe-style product modal viewer
- Category filters & live search
- Floating WhatsApp button
- Premium dark theme with purple neon glow
- 100% mobile optimized

## 🛠️ Tech Stack
- HTML5 + Vanilla CSS + Vanilla JavaScript
- No frameworks, no dependencies — ultra fast!

## 📦 Structure
```
├── index.html
└── assets/
    ├── css/style.css
    ├── js/
    │   ├── products.js   ← Edit products here
    │   └── app.js
    └── images/
```

## ⚙️ Adding Products
Edit `assets/js/products.js` — add a new object to the `PRODUCTS` array.

## 📱 WhatsApp Number
Change your number in `assets/js/products.js`:
```js
whatsappNumber: 'YOUR_NUMBER_HERE'
```
